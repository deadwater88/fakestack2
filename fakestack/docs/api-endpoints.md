# API Endpoints

## HTML API

### Root
- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users/:userId`
- `GET /api/users/:userId`

### relevantUsers
- `GET /api/users/:userId/relevant_users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Comments
- `GET /api/comments/:comment_id`
- `POST /api/comments/`
- `PATCH /api/comments/:comment_id/`
- `DELETE /api/comments/:comment_id/`
